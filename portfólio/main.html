Gym Backend

Gym Backend

Nesse projeto utilizei conceitos de AdonisJS para gerar uma API que realiza as operações de CRUD. Essa API tem como fundamento catalogar sessões de treinamentos. É uma API para mais dois amigos, onde eles podem adicionar um treino nos seus dados específicos. Cada um só pode adicionar o seu próprio, ou seja, o acesso é limitado. O Frontend foi feito por meio de um bot no Telegram, onde eu utilizei Python para programá-lo. O bot automaticamente recebe como parâmetro o usuário do Telegram e realiza as requisições (GET, POST) dependendo do comando solicitado.

Abra a imagem em outra guia ou baixe para visualizar melhor.

A desvantagem é que a API só funciona localmente, o que exige ativação manual para ser utilizada. Entretanto, como é de pouco uso semanal, o trabalho não é dificultoso.
